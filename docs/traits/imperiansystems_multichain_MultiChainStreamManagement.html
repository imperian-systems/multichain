public

  • create() — Creates a new stream on the blockchain called 'name'.
  • createfrom() — This works like create, but with control over the from-address used to create the stream.
  • getstreamitem() — Retrieves a specific item with txid from stream, passed as a stream name, ref or creation txid, to which the node must be subscribed.
  • gettxoutdata() — Returns the data embedded in output vout of transaction txid, in hexadecimal.
  • liststreamblockitems() — This works like liststreamitems, but listing items within the specified blocks only.
  • liststreamitems() — Lists items in stream, passed as a stream name, ref or creation txid.
  • liststreamkeyitems() — This works like liststreamitems, but listing items with the given key only.
  • liststreamkeys() — Provides information about keys in stream, passed as a stream name, ref or creation txid.
  • liststreams() — Returns information about streams created on the blockchain.
  • publish() — Publishes an item in stream, passed as a stream name, ref or creation txid, with key provided in text form and data in hexadecimal.
  • publishfrom() — Publishes an item in stream, passed as a stream name, ref or creation txid, with key provided in text form and data in hexadecimal.
  • subscribe() — Instructs the node to start tracking one or more asset(s) or stream(s).
  • unsubscribe() — Instructs the node to stop tracking one or more asset(s) or stream(s).
